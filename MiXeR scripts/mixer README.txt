----Overview of MiXeR files-----
Script adapted from: https://github.com/precimed/mixer

format_sumstats.txt: script to preprocess GWAS sumstats in format compatible with MiXeR
mixer_univariate.sh:  executes univariate MiXeR analysis
mixer_bivaraite.sh:  executes bivariate  MiXeR analysis
mixer_figs.txt: generates figures from analysis
