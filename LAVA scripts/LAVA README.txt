----Overview of LAVA files-----
Script adapted from: https://surfdrive.surf.nl/files/index.php/s/rtmNm8YZERGwl7f


lava_rg.sh: job submission script to HPC cluster
lava_rg.job: executes analysis script and copies results into local directory
lava_rg.R: R script to perform local genetic correlation analysis for a phenotype pair
pheno.pairs.txt: lists all phenotype pairs for bivariate local correlation analysis 
settings.sh: contains list of directories needed for LAVA analysis
